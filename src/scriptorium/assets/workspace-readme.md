# Scriptorium synthetic workspace

This directory was generated by `scriptorium demo` from synthetic fixtures.

- `Projects/` is user-owned Markdown in a real workspace.
- `Reviews/` contains the Steward-generated literature review.
- `Reports/` contains Provenance search and MCP evidence.
- The demo uses a recorded agent draft fixture and makes no live model call.
- Obsidian, Zotero, credentials, and network access are not required.
